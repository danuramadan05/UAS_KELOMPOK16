## Deskripsi Proyek
Web ini terdiri dari backend sederhana menggunakan Express.js yang menyajikan file statis (HTML/CSS/JS) dan memiliki sebuah endpoint API sederhana.

## Cara Menjalankan Secara Lokal
1. Pastikan Node.js sudah terinstal.
2. Clone repository ini: `git clone [URL_GITHUB_KALIAN]`
3. Buka folder proyek dan jalankan `npm install`.
4. Jalankan server dengan perintah `node server.js`.
5. Buka browser dan akses `http://localhost:3000`.
